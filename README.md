# SkillBridge

SkillBridge is a web app that connects students with micro-internship opportunities from startups and nonprofits. It allows project posting, skill-based matching, and real-world experience building for early-career developers.

## 💻 Tech Stack

**Frontend:**
- React.js
- JavaScript (ES6+)
- Axios
- TailwindCSS (or Material UI)

**Backend:**
- Java (Spring Boot)
- Spring Security (JWT Auth)
- Hibernate (JPA)
- PostgreSQL

**Other Tools:**
- Docker (for deployment)
- Git & GitHub (version control)
- Postman (API testing)

## 📁 Folder Structure

```
skillbridge/
├── backend/       # Spring Boot code
```

## 🚀 Getting Started

**Frontend**
```bash
cd frontend
npm install
npm start
```

**Backend**
```bash
cd backend
./mvnw spring-boot:run
```
